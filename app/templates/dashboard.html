Welcome to the Test Management System

Welcome to your centralized hub for managing testing procedures and tests. This dashboard is designed to streamline the process of creating, reviewing, and analyzing tests and their procedures.

Getting Started

Start by following these simple steps:

  1. Define Procedures: Start by building some procedures for the tests. These procedures form the backbone of your testing framework, ensuring consistency and reliability.
  2. Create Tests: Once you have your procedures in place, you can create as many tests as you need for each procedure. This flexibility allows you to comprehensively test different scenarios under the same procedural umbrella.
  3. Review and Export: After conducting your tests, you can review and export everything for analysis. The system provides an easy way to revisit your data, allowing for continuous improvement and insights.

Important Notes

Please keep in mind:

  • Prototypes should be manually defined in the test structure JSON file.
  • While custom fields can be added using the procedure or test structure JSON files, essential fields such as ID, title, date, and attachments must not be removed.
